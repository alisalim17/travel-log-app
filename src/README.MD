# Surv

## Log Entry

- Title -> Text
- Description -> Text
- Start & End Date => Date
- Latitude => Number
- Longitude => Number
- Rating => 0-5
- Images => URL
- Created at => Date
- Post Id -> ID

## For start

- You must add your own firebase server account and config to utils file as serviceAccount.js and config.js

* `npm run dev` for start
